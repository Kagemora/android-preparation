# android-prep — сайт с вопросами к Android-собеседованиям

Статический сайт, без сборки, без зависимостей. 723 вопроса из 20 технических категорий + 2 справочные (Lifehacks, Resume), собранные из реальных собеседований (М-Видео, Т-Банк, СБЕР, Яндекс, Авито, Wildberries, Ozon, Rutube, ПСБ, Альфа-Банк, X5, Крипта и др.) и энциклопедий вопросов.

## Как захостить на GitHub Pages

1. Создай новый репозиторий на GitHub (можно назвать `android-prep`).
2. Положи все файлы из этой папки (`index.html`, `styles.css`, `app.js`, `categories.js`, `data.js`) **в корень** репозитория.
3. Закоммить и запушь:
   ```bash
   git init
   git add .
   git commit -m "android-prep site"
   git branch -M main
   git remote add origin https://github.com/Kagemora/android-prep.git
   git push -u origin main
   ```
4. В настройках репозитория: **Settings → Pages → Source: Deploy from a branch → Branch: main / (root)**.
5. Через минуту сайт будет доступен по адресу `https://<твой_ник>.github.io/android-prep/`.

## Структура

- `index.html` — разметка (сайдбар с категориями, дашборд, вид категории с поиском и списком вопросов)
- `styles.css` — тёмная тема
- `categories.js` — метаданные категорий (цвета, значки) + небольшой набор Lifehacks-вопросов
- `data.js` — 723 вопроса, сгенерировано из мастер-справочника (13 md-файлов). **Структуру полей менять не нужно** — только дозаполнять `answer.what` / `answer.diff` / `answer.example` у нужных вопросов.
- `app.js` — вся логика: рендер, поиск, аккордеон, прогресс

## Формат ответа на вопрос

У каждого вопроса в `data.js` есть поле:

```js
answer: { what: "", diff: "", example: "" }
```

- `what` — «Что это» (короткое определение своими словами)
- `diff` — «Ключевое отличие» (та самая деталь, которую реально проверяют на собесе)
- `example` — «Пример / следствие» (код или практическое следствие)

Пока эти поля пустые — карточка показывает вместо них нейтральную заглушку. Как только текст появится хотя бы в одном из трёх полей — на сайте автоматически отрисуются цветные подписи (синий/жёлтый/зелёный), без правки CSS/JS.

## Прогресс

Отметки "выучено" хранятся в `localStorage` браузера (ключ `android-prep-progress-v1`), то есть per-браузер, не синхронизируется между устройствами. Это осознанное упрощение — без бэкенда синхронизация невозможна.
