# android-preparation — сайт с вопросами к Android-собеседованиям

Статический сайт, без сборки, без зависимостей. 723 вопроса из 20 технических категорий + 2 справочные (Lifehacks, Resume), собранные из реальных собеседований (М-Видео, Т-Банк, СБЕР, Яндекс, Авито, Wildberries, Ozon, Rutube, ПСБ, Альфа-Банк, X5, Крипта и др.) и энциклопедий вопросов.

## Как зайти

Доступен по адресу `https://kagemora.github.io/android-preparation/`.
